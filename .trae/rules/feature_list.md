# 功能清单 - 精简版

## 10个功能模块

1. **WebSocket 连接模块** - 建立连接、自动重连
2. **消息接收与显示模块** - 接收消息、渲染卡片
3. **消息处理模块** - 去重检测、重点检测
4. **弹窗模块** - 显示重点消息弹窗
5. **图片处理模块** - 提取图片、防盗链、缓存
6. **复制功能模块** - 复制文字/图片
7. **上下文功能模块** - 显示历史消息
8. **窗口管理模块** - 窗口状态、置顶、最小化
9. **系统托盘模块** - 托盘图标、菜单
10. **设置模块** - 各种设置项

## 功能依赖关系图

```
WebSocket 连接模块
    ↓
消息接收与显示模块
    ↓
    ├── 消息处理模块（去重+重点）→ 弹窗模块
    ├── 图片处理模块 → 复制模块
    └── 上下文功能模块

窗口管理模块
    ├── 系统托盘模块
    └── 设置模块
```

## 危险操作列表

### 绝对不能随便改的
- 修改 `ws` 变量名（WebSocket）
- 修改 `savedMessages` 结构（消息存储）
- 修改 `globalMsgId`（消息ID）
- 修改 `duplicateTracker` 结构（去重）
- 修改 `PopupData` 结构（弹窗数据）
- 修改窗口状态存储结构
- 修改 localStorage 键名
- 修改事件名称
- 修改函数签名
- 修改数据结构

### 可以安全修改的
- 修改 UI 样式
- 添加日志输出
- 修改默认值（如重连次数、缓存大小）
- 添加新的关键词（重点检测）
- 调整价格阈值

## 修改前检查清单

1. 在本文档中找到要修改的功能
2. 查看功能的依赖关系
3. 识别会影响哪些功能
4. 确定需要测试哪些功能
5. 查看"危险操作"避免踩坑
